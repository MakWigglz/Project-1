#### Frontend AJAX Call to Both APIs ```html Link Python Flask and Java ``` ####The HTML creates two buttons: flaskBtn: Calls the Flask API at http://localhost:5000/api/flask. javaBtn: Calls the Java API at http://localhost:8080/api/java. jQuery is included to handle AJAX requests more easily. The $(document).ready function ensures the JavaScript code runs after the DOM is loaded. Clicking a button triggers an AJAX request using jQuery's $.get method: $.get makes a GET request to the specified URL. The callback function handles the response: data: Contains the data returned by the API (if any). status: Indicates the success or failure of the request. An alert message displays the received data and status. To run the frontend, open the HTML file in a browser and click the buttons to call the Flask and Java APIs.
